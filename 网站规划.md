
# 个人摄影网站技术规划（GitHub Pages 版本）

## 一、整体架构

### 方案：**纯静态网站 + GitHub Pages
- **框架**：Next.js 14 (Static Export) + React + TypeScript
- **样式**：Tailwind CSS
- **组件库**：shadcn/ui
- **数据存储**：JSON 文件 + Markdown 文件（本地存储在仓库中）
- **图片存储**：GitHub 仓库或免费图床（如 Imgur）
- **部署**：GitHub Pages / Vercel（可选）

---

## 二、功能模块设计

### 1. 首页模块
- 精选照片展示（网格）
- 最新心情/博客摘要
- 社交媒体链接入口
- 音乐播放器（悬浮/固定）

### 2. 相册模块
- 照片组列表（按时间倒序）
- 日历视图筛选
- 标签筛选系统
- 照片详情页（大图预览）

### 3. 心情/博客模块
- 文章列表（时间线）
- 文章详情页
- 支持 Markdown

### 4. 社交媒体关联模块
- 小红书作品关联卡片
- 微博作品关联卡片
- 手动添加（编辑 JSON 文件）
- 一键跳转

### 5. 音乐分享模块
- 固定几首歌曲展示
- 音乐播放器（使用 HTML5 Audio）

## 六、部署方案

### GitHub Pages 部署
1. 创建 GitHub 仓库
2. 使用 GitHub Actions 自动构建和部署
3. 访问地址：`https://你的用户名.github.io/blog/

### 部署配置
- 使用 `next build && next export 生成静态文件
- 推送到 GitHub
- GitHub Actions 自动部署
