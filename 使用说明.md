# 网站使用说明

## 如何新增相册、心情等内容

### 方法一：通过管理页面（推荐）

1. 在网站首页右上角点击 **⚙️ 管理**
2. 选择对应的数据类型（相册/心情/音乐/社交媒体）
3. 点击 **"复制 JSON"** 按钮
4. 在项目文件夹中打开对应的 JSON 文件（例如 `data/albums.json`）
5. 粘贴并编辑内容
6. 保存文件，网站会自动刷新显示更新

### 方法二：直接编辑 JSON 文件

#### 1. 新增相册

编辑 `data/albums.json`，添加新的相册对象：

```json
{
  "id": "your-album-id",
  "title": "相册标题",
  "description": "相册描述（可选）",
  "coverImage": "封面图片URL",
  "date": "2024-03-16",
  "tags": ["标签1", "标签2"],
  "photos": [
    {
      "id": 1,
      "title": "照片标题（可选）",
      "url": "照片URL",
      "tags": ["标签1"]
    }
  ]
}
```

#### 2. 新增心情/博客

编辑 `data/posts.json`，添加新的文章对象：

```json
{
  "id": "your-post-id",
  "title": "标题（可选）",
  "date": "2024-03-16",
  "isMood": true,
  "content": "文章内容"
}
```

- `isMood: true` = 心情
- `isMood: false` = 博客

#### 3. 新增音乐

编辑 `data/music.json`，添加新的音乐对象：

```json
{
  "id": 1,
  "title": "歌曲名",
  "artist": "歌手",
  "album": "专辑（可选）",
  "coverUrl": "封面图片URL（可选）",
  "url": "音乐文件URL",
  "platform": "netease"
}
```

#### 4. 新增社交媒体链接

编辑 `data/social-links.json`，添加新的链接对象：

```json
{
  "id": 1,
  "platform": "xiaohongshu",
  "title": "链接标题",
  "url": "https://...",
  "thumbnailUrl": "缩略图URL（可选）"
}
```

- `platform: "xiaohongshu"` = 小红书
- `platform: "weibo"` = 微博

## 图片来源建议

- 照片可以上传到 GitHub 仓库的 `public/images/` 文件夹
- 或者使用免费图床（如 Imgur）
- 或者使用其他图片托管服务

## 部署到 GitHub Pages

1. 将项目推送到 GitHub 仓库
2. 在仓库设置中开启 GitHub Pages
3. 选择分支和源文件夹
4. 网站将自动部署！

## 注意事项

- ID 必须唯一，不能重复
- 日期格式必须是 `YYYY-MM-DD`（例如 2024-03-16）
- 修改 JSON 文件后，网站会自动热更新
