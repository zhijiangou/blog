# 我的个人网站

一个记录照片、心情和音乐的静态网站，部署在 GitHub Pages 上。

## 技术栈

- Next.js 14 (Static Export)
- TypeScript
- Tailwind CSS
- shadcn/ui

## 开始使用

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

## 数据管理

所有数据都存储在 `data/` 目录下的 JSON 文件中：

- `albums.json` - 相册数据
- `posts.json` - 心情/博客数据
- `social-links.json` - 社交媒体链接
- `music.json` - 音乐数据

## 部署

使用 GitHub Pages 或 Vercel 部署。
